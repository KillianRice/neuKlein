README FOR SOFTWARE/VERSION:  
Trouble.exe - Application to find out problems.

FOR PCO PRODUCT/VERSION:
PCI-Interface-Board 520xP
PCI-Interface-Board 525xP
PCI-Interface-Board 540xP
SensiCam, Dicam Pro, HSFC Pro, PixelFly

DESCRIPTION:
This application can be used to find out the error source if your
camera does not work. Simply run trouble.exe and select the camera
and board you like to test. Some textfiles will be generated which will
help you and our support to get close to the reason for errors.
